# Vue 3 - MoonFish front-end

## How to run

### Running in development mode
```bash
npm run dev
```

### Run app on production
```bash
npm run start
```

### Drop DB and run app on production 
```bash
npm run start-drop
```

## Visualisation

**Home page** : http://localhost:8080
